{% blocktranslate %} Fredagscaféen er en studenterdrevet bar på Institut for Datalogi, Aarhus Universitet, som siden 1993 har været et socialt samlingspunkt for studerende og ansatte på instituttet. Vi er kendt for vores store udvalg af specialøl, der kontinuerligt udskiftes. Derudover har vi fokus på at have noget for enhver smag og prisklasse, og vi har altid et stort udvalg af sodavand, cider, shots, glutenfri øl og alkoholfrie alternativer. Vi er samtidig en af de eneste fredagsbarer, der fast har fadøl fra {% endblocktranslate %} Aarhus Bryghus {% bs_icon 'box-arrow-up-right' size='0.75em' %}. {% blocktranslate %} Tag endelig et kig på vores faste sortiment, {% endblocktranslate %} {% translate "her" context "På linket her" %}.

{% blocktranslate %} Fredagscaféen drives af en frivillig bestyrelse, der består af studerende på instituttet, som også kan ses {% endblocktranslate %} {% translate "her" context "På linket her" %}. {% blocktranslate %} Baren er non-profit, og overskuddet går til vedligeholdelse af baren og arrangementer for bartenderne. {% endblocktranslate %}

{% translate "Adfærdskodeks" %} #

{% blocktranslate %} Fredagsbaren er et socialt arrangement for indskrevne studerende på Aarhus Universitet. Formålet er at skabe et trygt og hyggeligt fællesskab for alle deltagere. {% endblocktranslate %}

{% blocktranslate %} Fredagsbaren er i udgangspunktet for indskrevne studerende ved Aarhus Universitet. Gæster udefra er også velkomne, så længe de også respekterer retningslinjerne for adfærd herunder, samt dem fra {% endblocktranslate %} {% translate "Institut for Datalogi" %}.

{% blocktranslate %} Alle deltagere skal respektere følgende grundlæggende retningslinjer: {% endblocktranslate %}
  • {% translate "Opføre sig respektfuldt overfor de andre deltagende gæster og de frivillige bartendere." %}
  • {% translate "Undlad støjende, truende eller generende adfærd." %}
  • {% translate "Vær opmærksom på grænser - diskriminerende, krænkende eller på anden måde uacceptabel adfærd tolereres ikke." %}
  • {% translate "Følg gældende lovgivning, herunder regler om alkohol, rusmidler og rygning." %}

{% blocktranslate %} Arrangørerne forbeholder sig ret til at: {% endblocktranslate %}
  • {% translate "Afvise eller bortvise personer, der ikke opfylder retningslinjerne for adfærd ovenfor." %}
  • {% translate "Nægte adgang til fremtidige arrangementer, hvis gentagne eller særligt grove overtrædelser finder sted." %}

{% blocktranslate %} Indskrevne studerende, der forbryder sig mod AU's ordensregler, kan desuden blive omfattet af disciplinære foranstaltninger i henhold til {% endblocktranslate %} {% translate "Aarhus Universitet's ordensregler" %} {% bs_icon 'box-arrow-up-right' size='0.75em' %}.

{% blocktranslate %} Henvendelser vedrørende ordensreglementet eller konkrete hændelser kan rettes til arrangørerne af fredagsbaren, eller direkte til bestyrelsen gennem {% endblocktranslate %} {{ BEST_MAIL }}, {% blocktranslate %} eller specifikke medlemmer, hvis kontaktoplysninger kan findes {% endblocktranslate %} {% translate "her" context "På linket her" %}.
{% blocktranslate %} Alle henvendelser vil blive behandlet fortroligt, og der vil blive taget hensyn til den enkeltes ønsker om anonymitet. Intet er for småt eller for stort til at blive taget op, og vi opfordrer alle til at henvende sig, hvis de oplever eller observerer noget, der strider imod ordensreglerne. {% endblocktranslate %}
